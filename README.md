# Chat-Bot AI
### Installation
##### 1) Clone the project.
```
Git clone https://github.com/CarSeP/chat-bot.git
```
##### 2) Environment variables.
Create an **.env** file in the root of the project, following the example of **.env.example**.
##### 3) Install dependencies.
```
npm i
```
##### 4) Run the project.
```
npm run dev
```
